My DevOps App

CI/CD with Docker, Kubernetes & ArgoCD

Welcome

This application demonstrates a full CI/CD pipeline using modern DevOps tools.

🐳 Docker

Containerises the application to ensure consistent environments across development and production.

Kubernetes

Orchestrates container deployment, scaling, and management within the cluster.

GitHub Actions

Automates the CI pipeline including linting, building, and pushing container images.

ArgoCD

Implements GitOps by automatically syncing Kubernetes manifests with the cluster state.

GHCR

GitHub Container Registry stores and manages versioned Docker images.
